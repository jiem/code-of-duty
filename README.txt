Compile with: ./build.sh or javac CodeOfDuty.java

Run with input="input.txt" and output="output.txt": java CodeOfDuty
Run with input={inputArg} and output={outputArg}: java CodeOfDuty {inputArg} {outputArg}
Run with input=random vector with mean value {M} and size {K} and output={outputArg}: java codeOfDuty -rand {M} {K} {outputArg}
